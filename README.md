# Image Enhance

Enhance the clarity of low-quality images using the power of AI.

Pull and run

```
pnpm install
```

Add the following variables to a `.env` file:

```
REPLICATE_API_KEY=
UPSTASH_REDIS_REST_URL=
UPSTASH_REDIS_REST_TOKEN=
```

Then run

```
pnpm run dev
```

to start the dev server.
